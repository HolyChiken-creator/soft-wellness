# Автономні Telegram-нагадування без AI

Soft Wellness не звертається до ChatGPT, OpenAI API або іншої генеративної моделі. Cloudflare Worker використовує лише локально написані фрази та дані, які користувач сам зберіг у налаштуваннях.

## Що надсилає бот

За замовчуванням:

- 08:00 — ранковий раціон і денна норма води;
- 10:30 — нагадування про воду;
- 13:00 — обідній раціон;
- 16:00 — друге нагадування про воду;
- 19:00 — вечірній раціон;
- 21:00 — м’яка пауза для уваги до ментального стану.

Час, часовий пояс і тексти можна змінити в застосунку.

## Звертання «пане» або «пані»

У профілі людина обирає:

- `Мужчина` — бот звертається `пане`;
- `Женщина` — бот звертається `пані`.

Визначення виконується звичайною умовою в коді, а не AI-аналізом імені або фотографії.

## Автоматична синхронізація раціону

У розділі `Профіль → Telegram-нагадування` увімкнена опція автоматичного раціону. Після додавання, зміни або видалення продукту із сьогоднішнього сніданку, обіду чи вечері застосунок надсилає оновлений план у Worker.

Якщо потрібен постійний раціон, автоматичну синхронізацію можна вимкнути та ввести тексти вручну.

## Підключення користувача

1. У застосунку відкрити `Профіль → Telegram-нагадування`.
2. Натиснути `Створити код`.
3. Відкрити бота та надіслати `/start КОД`.
4. Після підтвердження Cron Trigger перевіряє розклад кожні 5 хвилин.

Команди:

- `/status`
- `/pause`
- `/resume`
- `/unlink`

Telegram не дозволяє боту самостійно почати приватну розмову, тому користувач має один раз відкрити бота та надіслати `/start КОД`.

## Секрети

Для GitHub Actions потрібні:

- `CLOUDFLARE_ACCOUNT_ID`
- `CLOUDFLARE_API_TOKEN`
- `TELEGRAM_BOT_TOKEN`

Workflow сам генерує секрет webhook, визначає username бота і завантажує всі Telegram-значення в Cloudflare Worker Secrets.
