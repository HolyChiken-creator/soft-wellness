# Soft Wellness 1.2

Mobile-first PWA для харчування, води та м’якої турботи про себе. Щоденник працює локально й офлайн. Опціональний Telegram-бот надсилає автономні нагадування через Cloudflare Workers без AI.

## Функції

- привітальний екран із затвердженим персонажем — дівчиною з чашкою серед рослин;
- анкета: ім’я, стать для звертання, вік, зріст і вага;
- автоматичне звертання бота `пане` або `пані` за пунктом, обраним у профілі;
- калорії = вага × 27;
- білок = вага × 2;
- жири = вага × 0,8;
- вуглеводи = залишок калорій ÷ 4;
- орієнтир води = вага × 30 мл, із можливістю вказати власну ціль;
- кнопки води `+250`, `+500`, довільний об’єм і скасування останньої порції;
- щоденник за датами: сніданок, обід, вечеря та перекус;
- каталог продуктів, грамування, КБЖВ, власні продукти, редагування й видалення;
- пошук штрихкоду через Open Food Facts;
- звіти, середня вода, історія ваги та серія ведення щоденника;
- локальна коротка позначка ментального стану;
- експорт та імпорт JSON;
- PWA manifest, Service Worker і Telegram Mini App API.

## Telegram без AI

Після одноразового підключення бот працює автономно:

- вранці, вдень і ввечері надсилає раціон;
- нагадує про воду;
- увечері надсилає коротку безпечну паузу для уваги до свого стану;
- використовує лише заздалегідь написані фрази;
- автоматично синхронізує сьогоднішній раціон після змін у щоденнику;
- зберігає мінімальні налаштування у SQLite-backed Durable Object.

Локальні записи щоденника не завантажуються на сервер. У Worker передається лише те, що потрібно для нагадувань: звертання, часи, ціль води та текст раціону.

## Деплой

Додайте в GitHub Actions Secrets:

1. `CLOUDFLARE_ACCOUNT_ID`
2. `CLOUDFLARE_API_TOKEN`
3. `TELEGRAM_BOT_TOKEN`

Після push у `main` workflow сам:

- перевіряє проєкт;
- генерує webhook secret;
- отримує username бота;
- створює Cloudflare Worker secrets;
- розгортає PWA та Worker;
- налаштовує Telegram webhook;
- перевіряє health endpoint.

Інструкція: [`docs/DEPLOY.md`](docs/DEPLOY.md).

## Приватність і обмеження

Токен Telegram ніколи не повинен бути в клієнтському JavaScript або публічному репозиторії. Він передається GitHub Actions у Cloudflare як encrypted Worker secret.

Ціль води та формула харчування є налаштовуваними орієнтирами, а не медичним призначенням. Нагадування про ментальний стан не є діагностикою або заміною професійної допомоги.
